# Pan Żelek - Порівняння дизайнів

React додаток для порівняння трьох варіантів дизайну сайту Pan Żelek.

## Варіанти дизайну

1. **Default** - Яскравий, веселий дизайн з рожево-жовтою темою
2. **Craft** - Крафтова, тепла тема з коричневими/земляними кольорами
3. **Premium** - Мінімалістичний, елегантний дизайн у чорно-білій гамі

## Встановлення та запуск

### Встановлення залежностей

```bash
npm install
```

### Запуск у режимі розробки

```bash
npm run dev
```

Додаток буде доступний за адресою `http://localhost:5173`

### Збірка для продакшну

```bash
npm run build
```

### Попередній перегляд збірки

```bash
npm run preview
```

## Деплой на GitHub Pages

### Важливо: Налаштування base path

Перед деплоєм переконайтеся, що в `vite.config.ts` вказано правильний `base`:

- Якщо ваш репозиторій називається `mytest`, залиште `base: '/mytest/'`
- Якщо репозиторій називається інакше, змініть на відповідне значення
- Якщо використовуєте custom domain або репозиторій називається як `username.github.io`, змініть на `base: '/'`

### Кроки для деплою:

1. **Створіть репозиторій на GitHub** (якщо ще не створено)

2. **Ініціалізуйте git та зробіть перший commit:**

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git push -u origin main
```

3. **Деплой на GitHub Pages:**

```bash
npm run deploy
```

Ця команда:
- Збере проект (`npm run build`)
- Створить/оновить гілку `gh-pages` зі зібраними файлами
- Запушить зміни на GitHub

4. **Увімкніть GitHub Pages в налаштуваннях репозиторію:**

- Перейдіть в Settings → Pages
- У розділі "Source" виберіть `gh-pages` branch
- Збережіть зміни

5. **Дочекайтеся деплою** (зазвичай 1-2 хвилини)

Ваш сайт буде доступний за адресою:
`https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`

## Використання

Після відкриття сайту вгорі з'явиться перемикач з трьома кнопками:
- **Default** - перемикає на яскравий дизайн
- **Craft** - перемикає на крафтову тему
- **Premium** - перемикає на мінімалістичний дизайн

Клієнт може легко перемикатися між варіантами та порівнювати їх.

## Технології

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React (іконки)

## Структура проекту

```
mytest/
├── src/
│   ├── components/
│   │   ├── PanZelekDefault.tsx  # Default дизайн
│   │   ├── PanZelekCraft.tsx     # Craft дизайн
│   │   └── PanZelekPremium.tsx   # Premium дизайн
│   ├── App.tsx                    # Головний компонент з перемикачем
│   ├── main.tsx                  # Точка входу
│   └── index.css                 # Глобальні стилі
├── index.html
├── package.json
├── vite.config.ts
└── tailwind.config.js
```

## Примітки

- Всі три дизайни використовують однакові мок-дані для товарів
- Кожен дизайн має власний стиль та колірну схему
- Перемикання між дизайнами відбувається миттєво без перезавантаження сторінки
